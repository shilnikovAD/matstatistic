# T8. Оптимальная оценка vs метод моментов для $U[0,\theta]$

## Что делает скрипт
Берём $\theta \sim U[1, 10]$, генерируем 101 выборку размера $n = 1000$ из $U[0, \theta]$ и в каждой считаем две оценки:
- $\hat\theta_1 = \frac{n+1}{n} X_{(n)}$ — поправленная ОМП (она же UMVUE);
- $\hat\theta_2 = 2\bar X$ — оценка методом моментов.

Строятся 4 гистограммы (сами оценки и их масштабированные версии — $n(\hat\theta_1 - \theta)$ и $\sqrt n(\hat\theta_2 - \theta)$), и эксперимент повторяется 3 раза для разных $\theta$. Заодно скрипт печатает эмпирические SD и сравнивает их с теоретическими.

## Теория

### Откуда берутся оценки

**ОМП.** Правдоподобие для $U[0,\theta]$ это

$$L(\theta) = \frac{1}{\theta^n}\,\mathbf 1\{X_{(n)} \le \theta\},$$

то есть оно убывает по $\theta$ при $\theta \ge X_{(n)}$, и максимум достигается в точке $\theta = X_{(n)}$. Только эта ОМП смещена: $\mathbb E X_{(n)} = \frac{n}{n+1}\theta$, поэтому правят коэффициентом: $\hat\theta_1 = \frac{n+1}{n} X_{(n)}$. По теореме Лемана–Шеффе $X_{(n)}$ полна и достаточна, поэтому $\hat\theta_1$ — UMVUE (несмещённая с минимальной дисперсией).

**Метод моментов.** Приравниваем $\bar X$ к $\mathbb E X = \theta/2$ — получаем $\hat\theta_2 = 2\bar X$. Тоже несмещённая.

### Дисперсии

Распределение $X_{(n)}$ имеет плотность $g(y) = n y^{n-1}/\theta^n$ на $[0, \theta]$. Считаем моменты:

$$\mathbb E X_{(n)} = \frac{n}{n+1}\theta,\qquad \mathbb E X_{(n)}^2 = \frac{n}{n+2}\theta^2,$$

откуда $\mathrm{Var}(X_{(n)}) = \frac{n\theta^2}{(n+1)^2(n+2)}$ и

$$\mathrm{Var}(\hat\theta_1) = \left(\frac{n+1}{n}\right)^2 \mathrm{Var}(X_{(n)}) = \frac{\theta^2}{n(n+2)}.$$

Для метода моментов всё проще: $\mathrm{Var}(X) = \theta^2/12$, поэтому

$$\mathrm{Var}(\hat\theta_2) = 4 \cdot \frac{\theta^2/12}{n} = \frac{\theta^2}{3n}.$$

Отношение:

$$\frac{\mathrm{Var}(\hat\theta_2)}{\mathrm{Var}(\hat\theta_1)} = \frac{n+2}{3} \to \infty.$$

Получается, что $\hat\theta_1$ убывает по скорости $1/n^2$, а $\hat\theta_2$ — стандартной $1/n$. С ростом $n$ разрыв в эффективности неограниченно растёт.

### А как же Рао–Крамер?

Если посчитать информацию Фишера формально, получим $I(\theta) = 1/\theta^2$, тогда «граница» Рао–Крамера для несмещённой оценки была бы $\theta^2/n$. Но $\mathrm{Var}(\hat\theta_1) = \theta^2/(n(n+2))$ — **меньше** этой границы. Что не так?

Ответ: условия Рао–Крамера здесь **не выполнены**. Стандартная регулярность требует, чтобы носитель распределения не зависел от параметра — а у нас носитель $[0,\theta]$ как раз зависит. В таких **нерегулярных** моделях возможна **сверхэффективность** — скорость $1/n^2$ вместо обычной $1/n$. Происходит это потому, что $X_{(n)}$ «прижимается» к $\theta$ на расстояние порядка $1/n$, а не $1/\sqrt n$.

В регулярных моделях такое невозможно: там для несмещённой оценки всегда $\mathrm{Var} \ge 1/(nI(\theta))$, то есть медленнее $1/n$ — нельзя.

### Предельные распределения

Метод моментов: $\sqrt n(\hat\theta_2 - \theta) \xrightarrow{d} \mathcal{N}(0, \theta^2/3)$ — обычная ЦПТ.

Оптимальная оценка: нормальности нет! Известно, что $\frac{n}{\theta}(\theta - X_{(n)}) \xrightarrow{d} \mathrm{Exp}(1)$, поэтому $n(\hat\theta_1 - \theta)$ имеет смещённое экспоненциальное предельное распределение — на гистограмме это выглядит как резко скошенная асимметричная фигура.

## Итог

| | $\hat\theta_1$ (UMVUE) | $\hat\theta_2$ (моменты) |
|---|---|---|
| Несмещённая? | да | да |
| Дисперсия | $\theta^2/(n(n+2))$ | $\theta^2/(3n)$ |
| Скорость | $1/n^2$ | $1/n$ |
| Предел | экспонента (после $n$) | нормальное (после $\sqrt n$) |
| Согласие с Рао–Крамером | нарушает (модель нерегулярна) | соблюдает |

Главное наблюдение: в нерегулярной модели $U[0,\theta]$ оптимальная оценка имеет дисперсию $\sim 1/n^2$ — это **быстрее** «универсальной» границы $1/n$. В регулярных моделях такого быть не может: лучшее, на что можно рассчитывать там — это $1/(nI(\theta))$.
