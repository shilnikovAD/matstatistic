# T7 - Оценивание параметра сдвига для N(theta, 1)

Это задание повторяет T6, но вместо распределения Коши используется нормальное
распределение N(theta,1). Сравниваются выборочная медиана и выборочное среднее.

## Постановка
1) Сгенерировать theta ~ U[-10,10].
2) Сгенерировать выборку X1,...,X1000 из N(theta,1).
3) Найти выборочную медиану theta_hat и выборочное среднее X_bar.
4) Повторить шаги 2-3 еще 100 раз, получив 101 значение theta_hat и 101 значение X_bar.
5) Построить гистограмму значений theta_hat и провести вертикальную линию при theta.
6) Построить гистограмму значений X_bar и провести вертикальную линию при theta.
7) Сделать вывод, какая статистика лучше оценивает theta.
8) Повторить шаги 1-7 еще 2 раза (итого 3 независимых запуска).
9-12) Сравнить теорию и эксперимент: состоятельность и асимпт. нормальность.

## Что делает скрипт
- Выбирает theta ~ U[-10,10].
- Генерирует 101 выборку размера n=1000 из N(theta,1).
- Считает выборочную медиану и среднее в каждой выборке.
- Строит гистограммы:
  - распределение theta_hat с линией theta;
  - распределение X_bar с линией theta;
  - распределение sqrt(n)*(theta_hat-theta);
  - распределение sqrt(n)*(X_bar-theta).
- Повторяет эксперимент 3 раза (для трех независимых theta).

## Теория (кратко, к пунктам 9-12)
- Для нормального распределения X_bar является МНП и МНК-оценкой theta,
  состоятельна и асимптотически нормальна:
  sqrt(n)*(X_bar-theta) -> N(0,1).
- Медиана theta_hat тоже состоятельна и асимптотически нормальна:
  sqrt(n)*(theta_hat-theta) -> N(0, 1/(4 f(theta)^2)).
  Для нормального распределения f(theta)=1/sqrt(2*pi), поэтому дисперсия
  равна pi/2.
- В нормальном случае среднее эффективнее (меньшая асимптотическая дисперсия),
  поэтому оценивает theta лучше, чем медиана.

## Выходные файлы
PNG-гистограммы сохраняются в папку `out` (или в другую, указанную через `--outdir`).
Имена вида:
- `run_1_median.png`, `run_1_mean.png`
- `run_1_median_scaled.png`, `run_1_mean_scaled.png`

## Запуск
```powershell
pip install -r ..\requirements.txt
python main.py
```

## Параметры
- `--n` размер выборки (по умолчанию 1000)
- `--reps` число повторов в одном запуске (по умолчанию 101)
- `--runs` число независимых запусков (по умолчанию 3)
- `--seed` фиксировать генератор случайных чисел
- `--outdir` папка для PNG
- `--show` показать окна с графиками
