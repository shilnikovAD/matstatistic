# T9. Две гирьки: F-критерий и оптимальные оценки

## Условие
Есть две гирьки с весами $\theta_1 = 5$ и $\theta_2 = 8$. На одних и тех же весах:
- $n$ раз взвешиваем первую — получаем $X_1, \dots, X_n$;
- $2n$ раз взвешиваем вторую — получаем $Y_1, \dots, Y_{2n}$;
- $n$ раз кладём обе — получаем $Z_1, \dots, Z_n$.

Шум во всех взвешиваниях независимый, $\sigma^2 = 1$.

## Что делает скрипт
1. Для каждого $n = 1, \dots, 200$ моделирует эксперимент, считает МНК-оценки и проверяет две гипотезы F-критерием:
   - $H_0: 8\theta_1 = 5\theta_2$ (на самом деле верна, $40 = 40$);
   - $H_0: 2\theta_1 = 3\theta_2$ (неверна, $10 \neq 24$).
2. Для каждой гипотезы строит график накопительной доли «принятий» $\bar\xi_n = \frac{1}{n}\sum_{i=1}^n \xi_i$, где $\xi_i = 1$, если $H_0$ не отвергнута, и 0 иначе.
3. Сравнивает оптимальные оценки $\hat\theta_1, \hat\theta_2$ с наивными $\bar X = \sum X_i/n$ и $\bar Y = \sum Y_j/(2n)$ — и траекторно по $n$, и через Monte Carlo (2000 повторов при $n = 200$).

## Теория

### Модель в матричном виде

Пишем $Y = M\beta + \varepsilon$, где $\beta = (\theta_1, \theta_2)^\top$, $\varepsilon \sim \mathcal{N}(0, I_{4n})$, а матрица плана состоит из трёх блоков:

$$M = \begin{pmatrix} \mathbf 1_n & \mathbf 0_n \\ \mathbf 0_{2n} & \mathbf 1_{2n} \\ \mathbf 1_n & \mathbf 1_n \end{pmatrix}.$$

Прямым счётом:

$$M^\top M = \begin{pmatrix} 2n & n \\ n & 3n \end{pmatrix},\quad \det = 5n^2,\quad (M^\top M)^{-1} = \frac{1}{5n}\begin{pmatrix} 3 & -1 \\ -1 & 2 \end{pmatrix}.$$

### Оптимальные оценки

При нормальном шуме МНК = ОМП = BLUE. Считая $S_X = \sum X_i$, $S_Y = \sum Y_j$, $S_Z = \sum Z_k$:

$$\hat\theta_1 = \frac{3 S_X - S_Y + 2 S_Z}{5n},\qquad \hat\theta_2 = \frac{-S_X + 2 S_Y + S_Z}{5n}.$$

Дисперсии берутся из диагонали $(M^\top M)^{-1}$:

$$\mathrm{Var}(\hat\theta_1) = \frac{3}{5n},\qquad \mathrm{Var}(\hat\theta_2) = \frac{2}{5n}.$$

### Сравнение с наивными оценками

Если игнорировать совместные взвешивания $Z$:

$$\mathrm{Var}(\bar X) = \frac{1}{n},\qquad \mathrm{Var}(\bar Y) = \frac{1}{2n}.$$

Отношения:

$$\frac{\mathrm{Var}(\hat\theta_1)}{\mathrm{Var}(\bar X)} = \frac{3}{5} = 0.6,\qquad \frac{\mathrm{Var}(\hat\theta_2)}{\mathrm{Var}(\bar Y)} = \frac{4}{5} = 0.8.$$

То есть оптимальная оценка для $\theta_1$ в $5/3 \approx 1.67$ раза эффективнее $\bar X$, а для $\theta_2$ — в $1.25$ раза эффективнее $\bar Y$. Это «бесплатная» эффективность за счёт того, что $Z_k$ несут информацию сразу о $\theta_1 + \theta_2$.

### F-критерий

Гипотеза $H_0: a\theta_1 - b\theta_2 = 0$ — линейное ограничение $A\beta = 0$ с $A = (a, -b)$, $q = 1$. Считаем RSS полной модели и ограниченной (где остаётся один параметр после подстановки $\theta_2 = (a/b)\theta_1$), статистика:

$$F = \frac{(\mathrm{RSS}_r - \mathrm{RSS}_{\text{full}})/1}{\mathrm{RSS}_{\text{full}}/(4n - 2)} \sim F_{1,\,4n-2}\;\text{ под } H_0.$$

Отвергаем при $p\text{-value} < \alpha$.

### Чего ждать в эксперименте

Если $H_0$ верна (для $8\theta_1 = 5\theta_2$): вероятность отвергнуть равна ровно $\alpha = 0.05$, поэтому $\bar\xi_n \to 1 - \alpha = 0.95$.

Если $H_0$ неверна (для $2\theta_1 = 3\theta_2$): мощность критерия растёт линейно по $n$, и доля «принятий» стремительно падает к нулю.

## Что получилось

### Численные результаты (seed=42)

| | Эмпирика | Теория |
|---|---|---|
| $\bar\xi_{200}$, $H_0: 8\theta_1=5\theta_2$ (верна) | 0.9350 | $1-\alpha = 0.95$ |
| $\bar\xi_{200}$, $H_0: 2\theta_1=3\theta_2$ (неверна) | 0.0000 | $\to 0$ |
| SD($\hat\theta_1$) при $n=200$ | 0.05439 | $\sqrt{3/1000} \approx 0.05477$ |
| SD($\bar X$) при $n=200$ | 0.07022 | $1/\sqrt{200} \approx 0.07071$ |
| SD($\hat\theta_2$) при $n=200$ | 0.04486 | $\sqrt{2/1000} \approx 0.04472$ |
| SD($\bar Y$) при $n=200$ | 0.04915 | $1/\sqrt{400} \approx 0.05$ |
| $\mathrm{Var}(\hat\theta_1)/\mathrm{Var}(\bar X)$ | 0.6000 | $3/5$ |
| $\mathrm{Var}(\hat\theta_2)/\mathrm{Var}(\bar Y)$ | 0.8331 | $4/5$ |

Получается почти идеальное согласие — особенно отношения дисперсий, которые совпали с теорией в 3-4 знака.

### Графики

- На графике доли для верной гипотезы синяя кривая колеблется около красной горизонтали $0.95$, амплитуда колебаний уменьшается как $1/\sqrt n$.
- Для неверной гипотезы доля очень быстро (уже при $n \sim 5$) падает до нуля — мощность F-критерия здесь очень высокая.
- На графиках траекторий оценок видно, что и оптимальная, и наивная сходятся к истине, но синяя (оптимальная) кривая заметно ýже разброса оранжевой (наивной).
- На гистограммах при $n=200$ оптимальный колокол уже наивного, причём для $\theta_1$ это виднее (там разница больше).

## Итог

Совместные взвешивания дают независимую информацию о $\theta_1 + \theta_2$ — а значит, отбрасывать их и брать просто среднее каждой серии неэффективно. МНК-оценка использует все три типа измерений и оказывается строго лучше. F-критерий работает как положено: для верной комбинации частота отвержений ровно $\alpha$, для неверной — мощность улетает в единицу.

## Параметры
- `--n-max` максимальный $n$ (200)
- `--theta1`, `--theta2` истинные значения (5 и 8)
- `--sigma` СКО шума (1.0)
- `--alpha` уровень значимости (0.05)
- `--mc-reps`, `--mc-n` параметры Monte Carlo для сравнения дисперсий (2000, 200)
- `--seed` фиксировать генератор
- `--outdir` куда складывать PNG
- `--show` показать окна
