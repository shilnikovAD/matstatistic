# T6 — Оценивание параметра сдвига для распределения Коши

Это задание моделирует оценивание параметра сдвига \(\theta\) для распределения Коши
и сравнивает выборочную медиану и выборочное среднее.

## Постановка
1) Сгенерировать \(\theta \sim U[-10,10]\).
2) Сгенерировать выборку \(X_1,\dots,X_{1000}\) из распределения Коши
   с плотностью \(p_\theta(x) = 1/(\pi(1+(x-\theta)^2))\).
3) Найти выборочную медиану \(\hat\theta\) и выборочное среднее \(\bar X\).
4) Повторить шаги 2–3 еще 100 раз, получив 101 значение \(\hat\theta\) и 101 значение \(\bar X\).
5) Построить гистограмму значений \(\hat\theta\) и провести вертикальную линию при \(\theta\).
6) Построить гистограмму значений \(\bar X\) и провести вертикальную линию при \(\theta\).
7) Сделать вывод, какая статистика лучше оценивает \(\theta\).
8) Повторить шаги 1–7 еще 2 раза (итого 3 независимых запуска).
9–12) Сравнить теорию и эксперимент: состоятельность и асимпт. нормальность медианы и среднего.

## Что делает скрипт
- Выбирает \(\theta \sim U[-10,10]\).
- Генерирует 101 выборку размера n=1000 из Коши.
- Считает выборочную медиану и среднее в каждой выборке.
- Строит гистограммы:
  - распределение \(\hat\theta\) с линией \(\theta\);
  - распределение \(\bar X\) с линией \(\theta\);
  - распределение \(\sqrt{n}(\hat\theta-\theta)\);
  - распределение \(\sqrt{n}(\bar X-\theta)\).
- Повторяет эксперимент 3 раза (для трех независимых \(\theta\)).

## Теория (кратко, к пунктам 9–12)
- Для Коши математическое ожидание не существует, поэтому \(\bar X\)
  не является состоятельной оценкой \(\theta\) и не имеет асимптотической нормальности.
- Медиана \(\hat\theta\) состоятельна для \(\theta\).
- Асимптотика медианы:
  \(\sqrt{n}(\hat\theta-\theta) \Rightarrow N(0, 1/(4 f(\theta)^2))\),
  где \(f\) — плотность. Для Коши \(f(\theta)=1/\pi\), значит дисперсия
  \(\pi^2/4\).
- На гистограммах \(\hat\theta\) ожидается концентрация около \(\theta\),
  а для \(\bar X\) — заметная нестабильность из‑за тяжелых хвостов.

## Выходные файлы
PNG‑гистограммы сохраняются в папку `out` (или в другую, указанную через `--outdir`).
Имена вида:
- `run_1_median.png`, `run_1_mean.png`
- `run_1_median_scaled.png`, `run_1_mean_scaled.png`

## Параметры
- `--n` размер выборки (по умолчанию 1000)
- `--reps` число повторов в одном запуске (по умолчанию 101)
- `--runs` число независимых запусков (по умолчанию 3)
- `--seed` фиксировать генератор случайных чисел
- `--outdir` папка для PNG
- `--show` показать окна с графиками
