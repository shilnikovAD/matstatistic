## Что делает скрипт
- Выбирает $\theta \sim U[-10,10]$.
- Генерирует 101 выборку размера $n=1000$ из Коши.
- Считает выборочную медиану и среднее в каждой выборке.
- Строит гистограммы:
  - распределение $\hat\theta$ с линией $\theta$;
  - распределение $\bar X$ с линией $\theta$;
  - распределение $\sqrt{n}(\hat\theta-\theta)$;
  - распределение $\sqrt{n}(\bar X-\theta)$.
- Повторяет эксперимент 3 раза (для трех независимых $\theta$).

### 1. Состоятельность

Оценка $T_n$ называется состоятельной для $\theta$, если $T_n \xrightarrow{P} \theta$ при $n\to\infty$.

**Среднее $\bar X_n$ для Коши.** Математическое ожидание не существует:

$$
\int_{-\infty}^{\infty}\frac{|x|}{\pi(1+x^2)}\,dx=\infty,
$$

поэтому ЗБЧ неприменим. Более того, у $\bar X_n$ для $\mathrm{Cauchy}(\theta, 1)$ **то же самое распределение, что у одного наблюдения** — снова $\mathrm{Cauchy}(\theta, 1)$. Это легко получить через характеристические функции:

$$
\varphi_{\bar X_n}(t)=\bigl[\varphi_X(t/n)\bigr]^n=\bigl[e^{i\theta t/n - |t|/n}\bigr]^n = e^{i\theta t-|t|}.
$$

Это знаменитый «парадокс»: усреднение **не уменьшает разброс**, $\bar X_n$ не сходится к $\theta$ ни в каком смысле. Поэтому $\bar X_n$ **не состоятельна**.

**Медиана $\hat\theta_n$.** Для любого распределения с непрерывной плотностью $f$, удовлетворяющей $f(\theta)>0$ в точке истинной медианы $\theta$, выборочная медиана состоятельна. Для Коши $f(\theta)=1/\pi>0$ $\Rightarrow$ медиана **состоятельна**.

### 2. Асимптотическая нормальность

**Среднее.** ЦПТ требует конечной дисперсии. У Коши её нет, поэтому никакого нормального предела для $\sqrt{n}(\bar X_n-\theta)$ **не существует**. Это масштабированное среднее остаётся Коши — с ростом $n$ тяжесть хвостов не уменьшается.

**Медиана (общая теорема Лемана).** Если плотность $f$ непрерывна в окрестности медианы $\theta$ и $f(\theta)>0$, то

$$
\sqrt{n}\,(\hat\theta_n - \theta) \xrightarrow{d} \mathcal{N}\!\left(0,\,\frac{1}{4\,f(\theta)^2}\right).
$$

Для $\mathrm{Cauchy}(\theta, 1)$: $f(\theta) = 1/\pi$, значит

$$
\mathrm{Var}_{\text{асимпт}}\bigl(\sqrt{n}(\hat\theta_n-\theta)\bigr)=\frac{1}{4\cdot(1/\pi)^2}=\frac{\pi^2}{4}\approx 2.467,\qquad
\mathrm{SD}=\frac{\pi}{2}\approx 1.571.
$$

При $n=1000$ ожидаемая SD медианы $\approx \pi/(2\sqrt{1000})\approx 0.0497$.

## Параметры
- `--n` размер выборки (по умолчанию 1000)
- `--reps` число повторов в одном запуске (по умолчанию 101)
- `--runs` число независимых запусков (по умолчанию 3)
- `--seed` фиксировать генератор случайных чисел
- `--outdir` папка для PNG
- `--show` показать окна с графиками
